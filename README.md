# Выявление зависимости цены смартфонов от их характеристик

В этом исследовании я провел глубокий анализ, чтобы выявить, насколько конкретные характеристики телефона, например, бренд, размер экрана или объем батареи, влияют на его цену. В ходе работы использовал знания эконометрики и Мат. статистики для построения аналитической модели, исправления проблем с данными (гетероскедастичность, автокорелляция, мультиколлинеарность и тд) и анализа результатов с определенной статистической значимостью! 

## Структура проекта

*   `Творческое задание.ipynb` — основной Jupyter Notebook с анализом.
*   `requirements.txt` — список используемых библиотек Python.
*   `.gitignore` — файл для исключения служебных файлов.


## Стек технологий

*   **Язык:** Python
*   **Библиотеки:** pandas, numpy, matplotlib, seaborn, statsmodels, scipy
*   **Инструменты:** Jupyter Notebook, Git

## Результаты

Результатом моей работы стала лог-нормальная модель, которая объясняет 51.1% вариации цен и имеет среднюю ошибку прогноза 40.4%. 
На основании моих данных и работы модели я выяснил, что больше всего на цену телефона влияет объяем RAM и размер экрана.
Эту информацию впослне можно использовать как покупателям смартфонов, так и крупным компаниям: люди понимают, за что они переплачивают, а производителям извесно, за что люди готовы платить!
